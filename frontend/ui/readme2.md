# Work Life Balance Analyzer

This project analyzes work life balance using questionnaires and interactive dashboards.

## Tech Stack

Frontend
- React
- Tailwind CSS
- Material UI
- Recharts
- Framer Motion

Backend
- Node.js / Flask (depending on your setup)

## Installation

Clone repository

git clone https://github.com/username/worklifebalance

Go to frontend folder

cd frontend/ui

Install dependencies

npm install

Install additional libraries

npm install recharts @mui/material @emotion/react @emotion/styled react-hook-form framer-motion react-icons react-gauge-chart axios

Run project

npm start

## Features

- Interactive questionnaire
- Work life balance score
- Trend charts
- Dashboard visualization
- Animated UI
